Reuse lichess mistake analysis

Based on [lichess' chessground](https://github.com/ornicar/chessground-examples) and [lichess API](https://lichess.org/api)


### Setup

```
nvm use v9.11.1
npm install gulp-cli -g
yarn install
yarn watch 
```
